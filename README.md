# Hello Thomas 🚂

A Rust project dedicated to learning the "Fortress" rules of systems programming.

### The Mission
- **Ownership & Borrowing:** Mastering the Librarian's rules.
- **Useful Engines:** Identifying and fixing "Diesel-style" logic errors.
- **Bullet Train Logic:** Exploring types and enums.

- **Now Featuring:** Synchronous, Command-Driven Communication across a small network of **Stations.**

- **Coming Soon:** Asynchronous, Decentralized, Event-Driven Architecture.

*"Either you're a Thomas, or you're a Diesel... or somewhere in between."*
